# Day 13 - Terraform Fundamentals

This lab covers the basic Terraform workflow and core concepts.

## Goal

Understand how Terraform works by creating and managing a simple local resource.

## Topics covered

- Infrastructure as Code
- Providers
- Resources
- Variables
- Outputs
- State
- Terraform workflow:
  - `terraform init`
  - `terraform fmt`
  - `terraform validate`
  - `terraform plan`
  - `terraform apply`
  - `terraform destroy`
